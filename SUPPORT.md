# Support

Need help with **Azokle Calculator**? We are here for you! 

Azokle Software is committed to providing a seamless, privacy-first experience. If you run into issues, please use the following channels to get assistance.

## 1. GitHub Issues
If you have found a bug or have a feature request, please open an issue on our [GitHub Issue Tracker](https://github.com/azoklesoftware/azokle-calculator-android/issues). 

Before opening a new issue, please:
- Search existing issues to see if your problem has already been reported.
- Ensure you are running the latest version of the app.

## 2. Direct Contact
For non-public inquiries, business related questions, or matters concerning Azokle Private Limited:
- **Website**: [https://azokle.com](https://azokle.com)
- **Support Email**: `support@azokle.com`

## 3. Security Vulnerabilities
If you have found a security vulnerability, **do not** open a public issue. Please read our [Security Policy](SECURITY.md) and report it directly to `security@azokle.com`.
