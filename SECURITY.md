# Security Policy

**Azokle Private Limited** and the **Azokle Software** division take the security of our applications and our users' privacy very seriously. Our core mission revolves around **Privacy-First Technology**.

## Supported Versions

We currently support the latest version of **Azokle Calculator** distributed via GitHub Releases and F-Droid.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Azokle Calculator, please do not disclose it publicly or open a public issue. Instead, please report it to us privately to ensure we can mitigate it before it affects our users.

You can report vulnerabilities by emailing:

**security@azokle.com**

Please include the following information in your report:
- A description of the vulnerability.
- Step-by-step instructions to reproduce the issue.
- Potential impact of the vulnerability.
- Any relevant logs or screenshots.

### What to Expect

- We will acknowledge receipt of your vulnerability report within 48 hours.
- We will investigate the issue and provide you with an estimated timeline for resolution.
- We will notify you when the vulnerability has been fixed and an update is available.
- We may acknowledge your contribution in our release notes (unless you prefer to remain anonymous).

Thank you for helping us keep **Azokle Calculator** safe and secure!
